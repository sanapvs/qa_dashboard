# QA Dashboard

A test visualization dashboard built with React + Vite + Tailwind CSS.

## Tech Stack
- React 18 (Vite)
- Tailwind CSS
- Mock data: JSON

## Running locally
npm install
npm run dev

## Week 1 progress
- Project setup and architecture defined
- Mock QA test data created (24 test cases, 7 modules)
- Basic dashboard UI with summary cards, trend chart, filter bar, and test table
